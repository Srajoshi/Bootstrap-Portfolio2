Portfolio:


Quisine:

Quisine: An app that allows users to search for recipes by cuisine, ingredient or recipe name. The search results provide links to individual recipes, the ingredients and nutrition information. Users can sign in using their google accounts to save recipes to their own separate page. Users can manage their saved recipes by adding or deleting.

The app also displays a list of restaurants near the user if geolocation is enabled.

Technologies used:
HTML, CSS, JavaScript, jQuery, geolocation, google authentication, Yelp API, Yummly API, Firebase.

neXt:

neXt : A travel app. The app provides users with the ability to plan their travel. Users are required to create an account using passport. Once logged in, users can create a trip and add various destinations. When a trip is saved, users can share the trip details via email. On trip completion, users can add reviews for the trip and based on their experience with food, tours, accommodations etc.

The front page of the app displays 10 most recent reviews of various destinations from our database. Users can also search for reviews for destinations.

Technologies used:
MVC framework, JavaScript, Node.js, Express, MySQL, Sequelize, Handlebars, passport, various npm packages.



Widget World:

Widget World: A MERN Stack Front to Back full stack application. This application allows users to create a personalized avatar and use voice commands to navigate to different widgets which include Weather, Calculator, Clock and Jokes. A trivia widget allows the user to test their knowledge in various categories and your avatar can also shop on Walmart website!

Technologies used:
MongoDB, Express, React, Node.js, passport, litmogies, SpeechRecognition, SpeechSynthesis, various npm packages, Trivia API, OpenWeatherMap API, Walmart API.
